# Shift Maker - シフト表作成アプリ

飲食店の店長向けのシフト表作成・管理アプリケーションです。

## 🚀 機能

- 📋 **データ入力**: CSVファイルからスタッフ情報を読み込み
- 📅 **シフト表生成**: 自動的に週間シフトを作成
- 📊 **ダッシュボード**: スタッフの勤務統計を表示
- ⚠️ **アラート機能**: スタッフ不足や相性問題を警告

## 技術スタック

- **フロントエンド**: Next.js 14 + TypeScript
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **ファイル処理**: PapaParse

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
